# e-shop-app
This project demonstrates the creation of a **responsive landing page** for an e-commerce website using **HTML, CSS, TailwindCSS**, and **vanilla JavaScript**.   The page is fully responsive across mobile, tablet, and desktop screens and includes interactive elements such as a **mobile navigation menu** and **form validation**.
## Features
- **Responsive Design:** Layout adapts to different screen sizes using TailwindCSS utility classes.
- **Hero Section:** Includes a heading, description, call-to-action button, and an image.
- **Products Section:** Showcases a grid of product cards with images, titles, descriptions, and "Buy Now" buttons.
- **About Section:** Highlights company information.
- **Contact Section:** Includes a contact form with client-side validation and a success message.
- **Mobile Menu Toggle:** Hamburger menu for mobile screens.
- **Interactive UI:** Hover effects on buttons and product images.

## Tech Stack
- **Languages:** HTML, CSS, JavaScript
- **CSS Framework:** TailwindCSS (CDN)
- **Interactivity:** Vanilla JavaScript for menu toggle and form validation
- **Responsive Design:** Mobile-first design principles
